    Content of the library¤

    Source¤

    The following files are used to build the current library:

    • Basis sets:

      • ALL_BASIS_SETS
      • BASIS_ADMM
      • BASIS_ADMM_MOLOPT
      • BASIS_ADMM_UZH
      • BASIS_MOLOPT
      • BASIS_MOLOPT_AcPP1
      • BASIS_MOLOPT_LnPP1
      • BASIS_MOLOPT_LnPP2
      • BASIS_MOLOPT_UCL
      • BASIS_MOLOPT_UZH
      • BASIS_ZIJLSTRA
      • BASIS_ccGRB_UZH
      • GTH_BASIS_SETS

    • Pseudopotentials:

      • AcPP1_POTENTIALS
      • GTH_POTENTIALS
      • LnPP1_POTENTIALS
      • LnPP2_POTENTIALS
      • POTENTIAL_UZH
